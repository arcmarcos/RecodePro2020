 - Nao era obrigatório o acesso remoto via SSH, mas eu fiz
 - Instalação e configuração do Apache, MySQL, PHP e PHPMyAdmin
 - Utilização do serviço NoIP
 - Permição e criação de usuário para acesso Remoto no MySQL
